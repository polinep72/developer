# T-Helper - AI помощник с векторным поиском

## Описание проекта
AI помощник с использованием FAISS для векторного поиска, предоставляющий интеллектуальную помощь пользователям.

## Основные компоненты
- **app.py** - основное приложение
- **faiss_index/** - векторный индекс FAISS
- **utils/** - утилиты для работы с векторами
- **static/** - статические файлы
- **templates/** - HTML шаблоны
- **Old/** - старые версии

## Функциональность
- Векторный поиск с использованием FAISS
- AI помощник для ответов на вопросы
- Веб-интерфейс для взаимодействия
- Обработка текстовых запросов
- Семантический поиск по базе знаний

## Технологии
- **Python** - основной язык
- **FAISS** - векторный поиск
- **Flask** - веб-фреймворк
- **HTML/CSS/JS** - фронтенд
- **Pickle** - сериализация данных

## Архитектура
```
T-Helper/
├── app.py              # Основное приложение
├── faiss_index/        # Векторный индекс
├── utils/              # Утилиты
├── static/             # Статические файлы
├── templates/          # HTML шаблоны
├── Old/               # Старые версии
└── *.pkl              # Сериализованные данные
```

## Текущий статус
- Проект функционален
- Использует FAISS для векторного поиска
- Имеет веб-интерфейс
- Включает систему индексации

## Известные проблемы
- Требуется настройка FAISS индекса
- Необходима настройка модели для векторизации
- Требуется настройка веб-интерфейса

## Ключевые настройки
- Путь к FAISS индексу
- Настройки модели векторизации
- Параметры поиска
- Настройки веб-интерфейса
