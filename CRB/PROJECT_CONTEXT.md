# CRB - Бот для работы с Центральным реестром банков

## Описание проекта
Telegram бот для работы с Центральным реестром банков (CRB), использующий APScheduler для планирования задач и PostgreSQL для хранения данных.

## Основные компоненты
- **crb_bot/bot_app.py** - основное приложение бота
- **crb_bot/config.py** - конфигурация
- **crb_bot/constants.py** - константы
- **crb_bot/handlers/** - обработчики команд
- **crb_bot/services/** - бизнес-логика
- **crb_bot/states/** - состояния пользователей
- **crb_bot/utils/** - утилиты
- **crb_bot/logs/** - логи приложения

## Функциональность
- Telegram бот для работы с банковскими данными
- Планировщик задач с APScheduler
- Работа с PostgreSQL базой данных
- Система состояний пользователей
- Логирование и мониторинг
- Обработка команд и сообщений

## Технологии
- **Python** - основной язык
- **Telegram Bot API** - интерфейс бота
- **APScheduler** - планировщик задач
- **PostgreSQL** - база данных
- **SQLAlchemy** - ORM для работы с БД
- **Logging** - система логирования

## Архитектура
```
CRB/
```

## Текущий статус
- Проект функционален
- Использует APScheduler с SQLAlchemyJobStore
- Интегрирован с PostgreSQL
- Имеет систему логирования
- Настроен для продакшена

## Известные проблемы
- Требуется настройка переменных окружения
- Необходима настройка PostgreSQL
- Требуется настройка Telegram Bot Token

## Ключевые настройки
- TELEGRAM_BOT_TOKEN - токен бота
- Параметры подключения к PostgreSQL
- Настройки планировщика APScheduler
- Таймзона для планировщика
- Настройки логирования
