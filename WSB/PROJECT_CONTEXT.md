# WSB - Бот для бронирования рабочих мест

## Описание проекта
Telegram бот для бронирования рабочих мест (WorkSpace Booking), использующий APScheduler для планирования задач и PostgreSQL для хранения данных.

## Основные компоненты
- **bot_app.py** - основное приложение бота
- **config.py** - конфигурация
- **constants.py** - константы
- **handlers/** - обработчики команд
- **services/** - бизнес-логика
- **states/** - состояния пользователей
- **utils/** - утилиты
- **wsb_bot/** - дополнительный модуль бота
- **heatmap_app/** - приложение для визуализации

## Функциональность
- Telegram бот для бронирования рабочих мест
- Планировщик задач с APScheduler
- Работа с PostgreSQL базой данных
- Система состояний пользователей
- Визуализация данных (heatmap)
- Логирование и мониторинг

## Технологии
- **Python** - основной язык
- **Telegram Bot API** - интерфейс бота
- **APScheduler** - планировщик задач
- **PostgreSQL** - база данных
- **Flask** - веб-интерфейс для heatmap
- **Logging** - система логирования

## Текущий статус
- Проект функционален
- Использует APScheduler для планирования
- Интегрирован с PostgreSQL
- Имеет систему логирования
- Включает веб-интерфейс для визуализации

## Известные проблемы
- Требуется настройка переменных окружения
- Необходима настройка PostgreSQL
- Требуется настройка Telegram Bot Token

## Ключевые настройки
- TELEGRAM_BOT_TOKEN - токен бота
- Параметры подключения к PostgreSQL
- Настройки планировщика APScheduler
- Таймзона для планировщика
- Настройки логирования
