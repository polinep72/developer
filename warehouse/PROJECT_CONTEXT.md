## Проект Soft_IPK / Склад готовой продукции

Этот корневой репозиторий содержит несколько подпроектов. Основной действующий подпроект сейчас — `crystal_wafer`, в котором развёрнуто веб‑приложение склада кристаллов/пластин и связанная инфраструктура.

- **Новый логический подпроект:** «Склад готовой продукции»
  - UI и работа с пользователями берутся за образец из `crystal_wafer/web_app_WebChip`
  - Основной экран — вкладка «Склад готовой продукции» на главной странице приложения с подвкладками:
    - «Микросхемы»
    - «Модули»
    - «ПП»
    - «Статистика/Аналитика/Графики»
  - Используется отдельная БД PostgreSQL `warehouse` с таблицами:
    - `invoice_ms`, `invoice_m`, `invoice_pp`
    - `consumption_ms`, `consumption_m`, `consumption_pp`

### Структура корневого проекта
- `crystal_wafer/` — основной монорепозиторий, внутри:
  - `web_app_WebChip/` — Flask‑приложение склада с авторизацией, управлением пользователями и UI вкладок
  - `telegram_bot_HMCB/` — телеграм‑бот
  - `backend/`, `frontend/` и вспомогательные файлы деплоя и документации
- `Склад готовой продукции.xlsx` — исходный файл‑описание требований к новому складу готовой продукции

### Текущий статус
