# Chip_and_Dip - Парсер остатков на складе

## Описание проекта
Веб-приложение для парсинга остатков товаров на складе сайта ChipDip с использованием Flask и Selenium WebDriver.

## Основные компоненты
- **app.py** - основное Flask приложение с веб-интерфейсом
- **parser_logic.py** - логика парсинга (импортируется)
- **templates/index.html** - веб-интерфейс для управления парсингом
- **chromedriver-win64/** - драйвер для Selenium

## Функциональность
- Веб-интерфейс для запуска парсинга
- Реальное время отслеживания прогресса через Server-Sent Events (SSE)
- Парсинг товаров из базы данных
- Обновление информации о товарах в БД
- Логирование всех операций

## Технологии
- **Flask** - веб-фреймворк
- **Selenium WebDriver** - автоматизация браузера
- **PostgreSQL** - база данных
- **Server-Sent Events** - реальное время обновлений
- **Threading** - фоновый парсинг

## Текущий статус
- Проект функционален
- Использует заглушки для БД (get_items_from_db_for_parsing, update_item_in_db)
- Настроен для работы с Chrome WebDriver
- Порт: 8085

## Известные проблемы
- Используются заглушки для работы с БД
- Необходима настройка реального подключения к БД
- Требуется настройка .env файла с конфигурацией

## Ключевые настройки
- FLASK_SECRET_KEY - секретный ключ Flask
- Конфигурация парсера загружается из parser_logic
- Headless режим настраивается через конфигурацию
- Таймауты и задержки настраиваются в parser_logic
