# Soft_IPK - Многоуровневый проект разработки

## Описание проекта
Soft_IPK - это комплексный проект, содержащий множество подпроектов для различных задач автоматизации, парсинга, ботов и веб-приложений. Проект организован как многоуровневая структура с независимыми подпроектами.

## Основные компоненты и подпроекты

### 1. Парсинг и автоматизация
- **Chip_and_Dip/** - парсер сайта Chip and Dip с веб-интерфейсом
- **ChipDip/** - расширенный парсер с Docker-контейнеризацией
- **Find_Chip/** - бот для поиска чипов
- **СИ/** - система интеграции с парсингом Excel файлов

### 2. Telegram боты
- **CRB/** - бот для работы с CRB (Центральный реестр банков)
- **WSB/** - бот для бронирования рабочих мест (WorkSpace Booking)
- **WSB v.3/**, **WSB v.4/** - версии бота бронирования
- **NewBot/** - новый бот с системой бронирования
- **telegram_bot_WorkSpaceBooking/** - основной бот бронирования
- **nl/** - бот с NLP возможностями
- **my_project/telegram_bot_HMCB/** - бот HMCB

### 3. Веб-приложения
- **Cristal/** - веб-приложение для работы с корзиной
- **dash/** - приложения на Dash
- **my_project/web_app_dash/** - веб-приложение на Dash
- **my_project/web_app_WebChip/** - веб-приложение WebChip
- **Zotero/** - веб-импортер для Zotero
- **zotero_web_importer_docker/** - Docker версия Zotero импортера
- **PDF2MD/** - конвертер PDF в Markdown

### 4. AI и машинное обучение
- **T-Helper/** - AI помощник с FAISS индексом
- **my_project/T-Helper/** - версия AI помощника
- **RAGFlow/** - полноценная RAG система
- **nl/** - NLP бот с базой данных

### 5. Системы управления
- **WorkControl/** - система контроля работы
- **my_project/backend/** - бэкенд система
- **my_project/frontend/** - фронтенд система

## Текущий статус проекта
- Проект активно развивается
- Множество независимых подпроектов
- Различные технологии: Python, Docker, Telegram API, веб-фреймворки
- Некоторые проекты имеют версионность (WSB v.3, v.4)

## Архитектура проекта
```
Soft_IPK/
├── Парсинг и автоматизация (Chip_and_Dip, ChipDip, СИ)
├── Telegram боты (CRB, WSB, NewBot, nl)
├── Веб-приложения (Cristal, dash, Zotero)
├── AI системы (T-Helper, RAGFlow)
└── Системы управления (WorkControl, backend/frontend)
```

## Связи между подпроектами

### Группы связанных проектов:

#### 1. Парсинг и автоматизация
- **Chip_and_Dip** ↔ **ChipDip**: Два парсера одного сайта с разными технологиями
- **СИ**: Система интеграции с парсингом Excel файлов
- **Find_Chip**: Бот для поиска чипов

#### 2. Telegram боты
- **CRB** ↔ **WSB** ↔ **NewBot**: Похожая архитектура ботов
- **WSB v.3** ↔ **WSB v.4**: Версии одного проекта
- **telegram_bot_WorkSpaceBooking**: Основной бот бронирования
- **nl**: NLP бот с базой данных

#### 3. AI системы
- **T-Helper** ↔ **RAGFlow**: AI системы с векторным поиском
- **my_project/T-Helper**: Версия T-Helper
- **nl**: NLP бот с AI возможностями

#### 4. Веб-приложения
- **Cristal** ↔ **dash**: Веб-приложения для работы с данными
- **Zotero** ↔ **zotero_web_importer_docker**: Версии Zotero импортера
- **my_project/web_app_dash** ↔ **my_project/web_app_WebChip**: Веб-приложения

#### 5. Системы управления
- **WorkControl**: Система контроля работы
- **my_project/backend** ↔ **my_project/frontend**: Бэкенд и фронтенд

### Общие технологии:
- **Python**: Основной язык для всех проектов
- **Docker**: Контейнеризация (ChipDip, RAGFlow, WorkControl)
- **PostgreSQL**: База данных (CRB, WSB, ChipDip)
- **Flask**: Веб-фреймворк (Chip_and_Dip, T-Helper, Cristal)
- **Telegram Bot API**: Боты (CRB, WSB, NewBot, nl)
- **APScheduler**: Планировщик задач (CRB, WSB)
- **FAISS**: Векторный поиск (T-Helper, RAGFlow)

## Известные проблемы
- Множество дублирующихся проектов (WSB v.3, v.4)
- Некоторые проекты могут иметь устаревшие зависимости
- Необходима унификация подходов к разработке

## Ключевые настройки
- Docker Compose для контейнеризации
- Python как основной язык разработки
- Telegram Bot API для ботов
- Веб-фреймворки для веб-приложений
- FAISS для векторного поиска в AI проектах
